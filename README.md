# Tetris Mobile

Versione mobile del classico gioco Tetris con supporto touchscreen, adattamento automatico allo schermo e migliorie rispetto alla versione originale.

## Caratteristiche

- Completamente compatibile con dispositivi touchscreen
- Adattamento automatico delle dimensioni alla risoluzione dello schermo (portrait)
- Sistema di controllo touch intuitivo
- Menu principale
- Sistema di livelli con aumento di difficoltà
- Preview del prossimo pezzo
- Salvataggio persistente degli high scores
- Effetti sonori
- Funzione pausa
